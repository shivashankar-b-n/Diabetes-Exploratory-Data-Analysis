# Diabetes Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Pima Indians Diabetes Dataset using Python.

## Dataset
- Pima Indians Diabetes Dataset
- Target Variable: Outcome

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## EDA Steps
- Data Loading
- Data Cleaning
- Missing Value Handling
- Descriptive Statistics
- Data Visualization
- Correlation Analysis

## Visualizations
- Count Plot
- Histogram
- Box Plot
- Correlation Heatmap
- Scatter Plot
- Pair Plot
- Violin Plot
- Pie Chart
- Bar Plot
- KDE Plot
- Line Plot
- BMI Histogram

## Conclusion
The analysis shows that Glucose, BMI, Age, and Pregnancies are important features related to diabetes prediction. The cleaned dataset is ready for machine learning model development.
